Features HaoZip Ru-Board Edition:


Included:

  • Russian (from RU-Board), English
  • Russian (from RU-Board), English SFX modules
  • After installation, shortcuts are created on the desktop and in the Start menu.
  • The program is correctly integrated into the system.
  • Uninstalling via installation / removal of programs

 

To install in a quiet mode, you can use the key

/S, /Type="X32" /Lng="English"
/S, /Type="X64" /Lng="English"
/S, /Type="X32" /Lng="Russian"
/S, /Type="X64" /Lng="Russian"
