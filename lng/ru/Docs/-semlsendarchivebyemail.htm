-seml (Send archive by email) ключ

Посылает архив по электронной почте.

Синтаксис

-seml[.]

  • [.]
  • Заставляет архив быть удаленным после присоединения его копии к сообщению электронной почты.

Пример

HaoZipC a archive.7z -seml a.txt

упаковывает файл a.txt и посылает его в виде archive.7z, используя электронную почту.

Команды, которые могут использоваться с этим ключом

a (Add), u (Update)
