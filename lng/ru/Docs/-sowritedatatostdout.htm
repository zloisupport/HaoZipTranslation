-so (write data to stdout) ключ

Заставляет HaoZip, записывать данные в stdout (стандартный поток вывода).

Синтаксис

-so

Примеры

HaoZipC x archive.gz -so > Doc.txt

распаковывает архив archive.gz в выходной поток и затем переадресовывает этот поток в файл Doc.txt.

HaoZipC a dummy -tgzip -so Doc.txt > archive.gz

упаковывает файл Doc.txt в стандартный выходной поток HaoZip и записывает этот поток в файл archive.gz.

Команды, которые могут использоваться с этим ключом

a (Add), e (Extract), u (Update), x (Extract with full paths)
