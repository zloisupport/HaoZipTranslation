-scc (Set charset for console input/output) ключ

Устанавливает кодировку для ввода/вывода в консоли.

Синтаксис

-scc{UTF-8 | WIN | DOS}

По умолчанию кодировка - DOS.

  • UTF-8
  • Устанавливает кодировку Unicode UTF-8.
  • WIN
  • Устанавливает кодировку, используемую Windows по умолчанию.
  • DOS
  • Устанавливает кодировку, используемую Windows в DOS (OEM) по умолчанию.

Пример

HaoZipC l archive.7z -sccUTF-8

выведет список файлов из archive.7z в кодировке UTF-8.
