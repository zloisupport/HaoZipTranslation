-slt (Show technical information) ключ

Устанавливает технический режим для команды l (List).

Синтаксис

-slt

Пример

HaoZipC l -slt archive.7z

показывает подробную техническую информацию о файлах в archive.7z.

Команды, которые могут использоваться с этим ключом

l (List)
