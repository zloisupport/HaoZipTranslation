-ssc (Set Sensitive Case mode) ключ

Устанавливает режим чувствительности к регистру символов для имен файлов.

Синтаксис

-scs[-]

Ключ

Описание

-ssc

Устанавливает режим с учетом регистра. Это - значение по умолчанию для Posix/Linux систем.

-ssc-

Устанавливает режим без учета регистра. Это - значение по умолчанию для Windows систем.

Пример

HaoZipC a archive.7z A*.txt -ssc -r

упаковывает все файлы A*.txt из текущего каталога и всех его - подкаталогов. Эта команда не упаковывает файлы a*.txt.

Команды, которые могут использоваться с этим ключом

a (Add), d (Delete), e (Extract), l (List), t (Test), u (Update), x (Extract with full paths)
