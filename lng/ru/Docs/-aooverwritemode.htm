-ao (Overwrite mode) ключ

Определяет режим замены в ходе извлечения для замены файлов, уже имеющихся на диске.

Синтаксис

-ao[a | s | t | u ]

Ключ

Описание

-aoa

Перезаписывает все существующие файлы без подсказки.

-aos

Пропустит извлечение существующих файлов.

-aou

Автоматически переименует извлекаемый файл (например, name.txt будет переименован в name_1.txt).

-aot

Автоматически переименует существующий файл (например, name.txt будет переименован в name_1.txt).

Пример

HaoZipC x test.zip -aoa

извлекает все файлы из архива test.zip и перезаписывает, существующее файлы без всякой подсказки.

Команды, которые могут использоваться с этим ключом

e (Extract), x (Extract with full paths)
