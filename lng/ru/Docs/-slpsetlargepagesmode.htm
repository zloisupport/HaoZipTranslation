-slp (Set Large Pages mode) ключ

Устанавливает режим больших страниц.

Синтаксис

-slp[-]

Ключ

Описание

-slp

Позволяет режим больших страниц.

-slp-

Отключает режим больших страниц. Эта опция - по умолчанию для всех команд.

Режим больших страниц увеличивает скорость сжатия. Однако, есть пауза в начале сжатия, в то время когда HaoZip распределяет большие страницы в памяти. Если HaoZip не может разместить большие страницы, он размещает обычные маленькие страницы. Кроме того, Диспетчер задач не показывает реальное использование памяти программами, если HaoZip используют большие страницы. Эта особенность работает только на Windows 2003 / XP x64. Также Вы должны иметь права администратора для вашей системы. Рекомендованный размер оперативной памяти для этой особенности - 1 Гб или больше. Чтобы установить эту особенность, Вы должны запустить HaoZip Менеджер файлов, по крайней мере, один раз, закрыть его и перезагрузить систему.

Ваша система может зависнуть на несколько секунд при запуске сжатия, если Вы используете режим -slp. Таким образом не рекомендуют использовать режим -slp, для сжатия небольших объемов данных (менее 100 Мб).

Пример

HaoZipC a archive.7z -slp a.iso

упаковывает файл a.iso с включенным режимом больших страниц.
