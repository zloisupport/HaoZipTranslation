-r (Recurse subdirectories) ключ

Определяет метод обработки подстановочных знаков и имен файла в командной строке.

Синтаксис

-r[- | 0]

Ключ

Описание

-r

Допустить рекурсию подкаталогов.

-r-

Отключить рекурсию подкаталогов. Эта опция - по умолчанию для всех команд.

-r0

Допустить рекурсию подкаталогов только для имен подстановочных знаков.

Примеры

HaoZipC l archive.zip *.doc -r-

составляет список всех *.doc файлов, которые принадлежат корневому каталогу архива в архиве archive.zip.

HaoZipC a -tzip archive.zip -r src\*.cpp src\*.h

добавляет все *.cpp и *.h файлы из каталога src и всех его подкаталогов в архив archive.zip.

HaoZipC a archive.7z folder1\

добавляет все файлы из каталога folder1 и всех его подкаталогов в архив archive.7z.

HaoZipC a archive.7z -r folder2\

ищет все каталоги folder2 во всех подкаталогах, и добавляет их (включая все подкаталоги) в архив archive.7z.

Команды, которые могут использоваться с этим ключом

a (Add), d (Delete), e (Extract), l (List), t (Test), u (Update), x (Extract with full paths)
