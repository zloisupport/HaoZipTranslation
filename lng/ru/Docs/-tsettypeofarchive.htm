-t (set Type of archive) ключ

Определяет тип архива.

Синтаксис

-t{archive_type}

  • {archive_type}
  • Определяет тип архива. Это может быть: 7z, zip, gzip, bzip2 или tar. 7z формат - значение по умолчанию.

Примечание: форматы gzip или bzip2 поддерживают только один файл в архиве. Если Вы хотите сжать больше чем один файл с этими форматами, создайте прежде tar архив, а затем сожмите его с вашим выбранным форматом.

Пример

HaoZipC a -tzip archive.zip *.txt

добавляет все *.txt файлы из текущего каталога в архив archive.zip.

HaoZipC t -t7z.split archive.7z.001

тестирует все файлы в archive.7z.001. Это также проверяет, что архив является многотомным .7z архивом.

HaoZipC x -tiso archive.iso

извлекает файлы из archive.iso, открывая его как ISO архив.

7z x -tudf archive.iso

извлекает файлы из archive.iso, открывая его как UDF архив.

Команды, которые могут использоваться с этим ключом

a (Add), d (Delete), e (Extract), l (List), t (Test), u (Update), x (Extract with full paths)
