-ssw (Compress files open for writing) ключ

Упаковывает файлы открытые для записи другим приложением. Если этот ключ не установлен, HaoZip не включает такие файлы в архив. 

Синтаксис

-ssw

Пример

HaoZipC a archive.7z -ssw *.txt

упаковывает все *.txt файлы в текущей папке, включая файлы открытые для записи другим приложениями.

Команды, которые могут использоваться с этим ключом

a (Add), u (Update)
