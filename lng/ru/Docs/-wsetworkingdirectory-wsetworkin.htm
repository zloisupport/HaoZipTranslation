-w (set Working directory) ключ

Устанавливает рабочий каталог для временного базового архива. По умолчанию, HaoZip строит новый базовый файл архива в том же самом каталоге, где и старый базовый файл архива. Определяя этот ключ, Вы можете установить рабочий каталог, где будет построен временный базовый файл архива. После того, как временный базовый файл архива построен, он копируется поверх первоначального; затем временный файл удаляется.

Синтаксис

-w[{dir_path}]

  • {dir_path}
  • Определяет путь каталога адресата. Не требуется, что бы путь заканчивался обратным слешем.

Если <dir_path> не назначен, то HaoZip будет использовать временный каталог Windows.

Пример

HaoZipC a -tzip archive.zip *.cpp -wc:\temp

добавляет *.cpp файлы в архив archive.zip, создавая временный архив в папке c:\temp.
