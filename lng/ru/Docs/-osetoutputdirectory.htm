-o (set Output directory) ключ

Определяет каталог адресат, куда файлы должны быть извлечены.

Этот ключ может использоваться только с командами извлечения.

Синтаксис

-o{dir_path}

  • {dir_path}
  • Это - путь каталога адресата. Он не обязательно должен заканчиваться обратным слешем. Если Вы определите * в {dir_path}, то HaoZip заменит этот символ * на название архива.

Пример

HaoZipC x archive.zip -oc:\Doc

извлекает все файлы из архива archive.zip в каталог c:\Doc.

HaoZipC x *.zip -o*

извлекает все *.zip архивы в подкаталоги с именами этих архивов.

Команды, которые могут использоваться с этим ключом

e (Extract), x (Extract with full paths)
