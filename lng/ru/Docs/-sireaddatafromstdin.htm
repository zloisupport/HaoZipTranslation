-si (read data from stdin) ключ

Заставляет HaoZip, читать данные из stdin (стандартный ввод) вместо дисковых файлов.

Синтаксис

-si{file_name}

  • {file_name}
  • Определяет имя, под которым будут сохранены в архиве сжатые данные. Если file_name не будет определено, то данные будут сохранены без имени.

Примечание: текущая версия HaoZip поддерживает чтение архивов из stdin только для xz, lzma, tar, gzip и bzip2 архивов.

Примеры

HaoZipC a archive.gz -tgzip -siDoc2.txt < Doc.txt

упаковывает входящий поток из файла Doc.txt в архив archive.gz, используя имя файла Doc2.txt.

HaoZipC x HaoZip4.tar.gz -so | HaoZipC x -si -ttar

распаковывает архив tar.gz.

Команды, которые могут использоваться с этим ключом

a (Add), e (Extract), u (Update), x (Extract with full paths)
