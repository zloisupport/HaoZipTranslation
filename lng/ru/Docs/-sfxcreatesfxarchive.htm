-sfx (Create SFX archive) ключ

Создает самораспаковывающийся архив.

Синтаксис

-sfx[{SFX_Module}]

  • {SFX_Module}
  • Определяет SFX модуль, который будет объединен с архивом. Этот модуль должен быть помещен в каталог рядом с HaoZipC.exe. Если {SFX_Module} не будет назначен, то HaoZip будет использовать стандартный SFX модуль.

SFX_Module

Описание

7zSetup_enu.sfx

Windows версия (английский язык).

7zCon_enu.sfx

Консольная версия.

7zSetup_rus.sfx

Windows версия (русский язык). В этом модуле можно менять Логотип и Иконку.

7zSetup_ukr.sfx

Windows версия (украинский язык). Можно менять Логотип и Иконку.

Min7zSetup_RUS.sfx

Windows версия (русский язык). В этом модуле нельзя менять Логотип и Иконку, потому что он сжат с помощью UPX.

Min7zSetup_ukr.sfx

Windows версия (украинский язык). Нельзя менять Логотип и Иконку - модуль сжат с помощью UPX.

Примеры

HaoZipC a -sfx a.exe *.txt

добавляет *.txt файлы в самораспаковывающийся архив a.exe, используя модуль по умолчанию.

HaoZipC a -sfx7zSetup_rus.sfx a.exe *

добавляют все файлы в самораспаковывающийся архив a.exe с модулем 7zSetup_rus.sfx, используя Windows версию SFX модуля.
